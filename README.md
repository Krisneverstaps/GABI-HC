This repository calculates cosmological luminosity distances using an analytic model and a neural network-based emulator. It also performs Bayesian inference on observational data using the bilby library and generates synthetic gravitational wave event data. This project looks to 
include CuPy and pymultinest, allowing GPU acceleration

Files
train_emulator.py: Trains the emulator
generate_synthetic_data.py: Generates synthetic gravitational wave event data
run_inference.py: Runs Bayesian likelihood inference
